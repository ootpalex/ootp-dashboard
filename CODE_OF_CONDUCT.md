# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1, as its code of conduct. The full text is available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

In short: contributors and maintainers commit to a welcoming environment for everyone regardless of background. Discussion stays focused on the project; disagreements stay civil.

## Reporting

If you experience or witness behavior that violates the Contributor Covenant, please report it by opening a private issue or contacting the project maintainer directly. Reports are reviewed in good faith and kept confidential.

## Scope

This applies to all project spaces — issues, pull requests, discussions, and any other forum the project uses — and to anyone representing the project in public spaces.
